# Physical AI & Humanoid Robotics Book

This repository contains the complete source for the "Physical AI & Humanoid Robotics" book, implemented as a Docusaurus project.

## About This Book

The Physical AI & Humanoid Robotics book is a comprehensive guide to building intelligent humanoid robots. It covers four critical modules:

1. **Module 1: ROS2 - The Robotic Nervous System** - Establishes the foundational communication and control framework
2. **Module 2: Digital Twin - Virtual Reality Bridge** - Explores simulation and digital representation technologies
3. **Module 3: Isaac - AI-Powered Robot Brain** - Delves into perception, planning, and decision-making systems
4. **Module 4: VLA - Voice-Enabled AI Robotics** - Integrates voice interfaces and advanced AI planning

## Installation

To run this documentation locally, you need Node.js version 18 or higher.

```bash
# Clone the repository
git clone <repository-url>
cd physical-ai-book

# Install dependencies
npm install

# Start the development server
npm start
```

The documentation will be available at `http://localhost:3000`.

## Building for Production

```bash
# Build the static site
npm run build
```

The built site will be in the `build` directory.
